DEMO

발펴고자는 사용자가 입력한 콘텐츠에
딥러닝 기술 (자연어처리&컴퓨터비전)을 입혀
완성된 PPT 파일을 제공합니다.

발펴고자의 주요 기능

발표 고자인 당신, 이젠 발 펴고 자!

발펴고자 AI가 단순반복 작업을 대신 수행하여 당신의 시간과 노력을 줄여드려요.

PPT 파일 생성

입력한 콘텐츠의 맥락을 파악하여 그와 어울리도록 PPT 파일을 만들어주고, 이를 다운로드할 수 있습니다.

이미지 배경 제거

컴퓨터 비전 기법을 통해 입력한 이미지의 배경을 제거해주고, 제거되는 정도는 직접 조절할 수 있습니다.

이미지 해상도 향상

저해상도 이미지를 딥러닝 모델을 통해 고해상도 이미지로 변환해주고, 돋보기 기능을 통해 변환 전후를 비교할 수 있습니다.

이미지의 픽토그램화

사전 학습된 알고리즘을 통해 실사 이미지를 픽토그램으로 변환해줍니다.

Computer Vision 기술 기반의

이미지 Background Removal 기능

1. 배경을 제거하려는 이미지를 업로드합니다. 2. START 버튼을 누르면 발펴고자 AI가 이미지를 딥러닝 처리합니다. 3. 처리가 끝난 이미지는 슬라이드바를 통해 배경이 제거되는 정도를 조절할 수 있습니다. 4. DOWNLOAD 버튼을 누르면 완성된 이미지를 다운로드할 수 있습니다.

배경 제거 기능 체험

Computer Vision 기술 기반의

이미지 Super Resolution 기능

1. 해상도를 높이려는 이미지를 업로드합니다. 2. START 버튼을 누르면 발펴고자 AI가 이미지를 딥러닝 처리합니다. 3. 돋보기를 통해 처리 전후 두 이미지의 해상도 차이를 확인힐 수 있습니다. 4. DOWNLOAD 버튼을 누르면 완성된 이미지를 다운로드할 수 있습니다.

해상도 향상 기능 체험

NLP 기술 기반의

PPT 자동 생성 기능

1. PPT로 만들고자하는 콘텐츠를 입력합니다. 필요하다면 이미지도 입력할 수 있습니다. 2. 변환하기 버튼을 누르면 발펴고자 AI가 맥락을 분석합니다. 3. 주제와 어울리는 템플릿, 슬라이드 별 어울리는 레이아웃으로 PPT를 완성합니다. 4. PPT 파일이 자동으로 다운로드됩니다.
